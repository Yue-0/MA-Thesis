# My MA Thesis LaTeX Source Code

这个仓库存储了我的硕士学位论文，目前正在撰写中，正文撰写进度会同步到本仓库。

## LaTeX 模板说明

由于[大连理工大学研究生院提供的LaTeX模版](https://gs.dlut.edu.cn/info/1210/13916.htm)问题很大，我根据[QuYue在OverLeaf上开源的模板](https://cn.overleaf.com/latex/templates/da-lian-li-gong-da-xue-shuo-shi-bi-ye-lun-wen-mo-ban-xelatex-v1-dot-0/mswbqtxykdff)进行了修改，主要做了以下调整：
* 添加了英文目录，
* 添加了常用符号表，
* 修改了原创性声明与授权书，与Word版本统一，
* 修改了其他细节，与Word版本一致。
